How to use

  • Select a tool (wall, start, end, erase) then click / drag on the grid.
  • Change Grid Density any time to reset and get a bigger / smaller canvas.
  • Press ▶ to run, ↻ to reset.

About the algorithms

Dijkstra’s explores every shortest path cost from the start.
A* adds a heuristic, making it dramatically faster in most cases.
BFS is unweighted and guarantees shortest path on grids.
The architecture is modular – drop additional algorithms into /scripts/algorithms and register them in main.js.
